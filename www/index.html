Freerails

Freerails is a real time multi player strategy game where players compete to build the most powerful railroad empire. It is written in java. Development is not yet complete.

 

Click here (~1.5MB) to play jfreerails 0.2.6 using Java webstart. (Requires Java 1.5)

If you are interested in getting involved in the Freerails project, post a message to the mailing list. Before writing any code, have a look at the coding guidelines.
