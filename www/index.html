FreeRails

FreeRails is a real-time, multiplayer railway strategy/management game where players compete to build the most powerful railroad empire. It is based on the RailRoad Tycoon I and II games. It is written in Java and the code is released under the GPL-2.0 license. Also see the history of FreeRails.
